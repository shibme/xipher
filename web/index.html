Share secrets, securely.

Xipher is a lightweight, no-install, server-free encryption tool. Everything happens right here in your browser, so your data never leaves your device unencrypted.

📄

Tip: open the 🔑 menu to set your own password or secret key so you can decrypt on any device.

Your key & password

Your identity is stored only in this browser. Set a password or paste an existing secret key to use the same identity across devices. Anyone who knows it can decrypt secrets sent to you, so keep it safe.

Use a long, memorable passphrase. The same password always produces the same identity.

A secret key starts with XSK_. Generate one with the CLI or the Random tab.

Generate a fresh, cryptographically random secret key. This replaces your current identity, so previously shared links will stop working for new secrets.
